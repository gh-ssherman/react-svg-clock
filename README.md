# Description
React SVG demo app by Stephen Sherman

<img src="static/assets/react-svg-clock.png">

# Usage

## Installation
```
npm i
```
## Run
```
npm run start
```
Go to `localhost:3000` in your browser

## Lint
```
npm run lint
```

## Test
```
npm run test
```

## Coverage
```
npm run test:coverage
```
